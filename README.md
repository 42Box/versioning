# 42Box data를 불러오기 위해서 사용되는 레포지토리입니다.

### deafult.json

- 폴더를 불러오기위해 사용합니다.
  - ex) 42Seoul 
    ``` json
    {
      "name": "42Seoul",
      "folder": "default-kr"
    }
    ```
    
### default-kr.json

- 해당 폴더의 URL들을 불러오기 위해 사용합니다.
  - ex) 42Seoul
      ``` json
      {
        "favorite": "https://profile.intra.42.fr/",
        "list": [
          {
            "name": "42 Intra",
            "url": "https://profile.intra.42.fr/"
          },
          {
            "name": "집현전",
            "url": "https://42library.kr/"
          },
          "_comment": "(..이하 생략)"
        ]
      }
      ```
